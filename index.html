Real Time Attendance System

The Real Time Attendance System is an innovative solution for automated attendance tracking using image processing techniques. It utilizes Python programming language and Haar Cascades for face detection and recognition. The system captures real-time images of individuals and compares them with stored datasets to mark attendance. This project involved extensive research, experimentation, and algorithm development to ensure accurate and reliable attendance tracking in various environments.

Duration: Feb 2023 - May 2023
